ATM Program – Case Study

Overview

The ATM Program is a console-based application developed in C#. It simulates a basic ATM system where users can view their account balance, withdraw money, and deposit funds securely.

Features

  • View Account Balance
  • Withdraw Funds
  • Deposit Funds
  • Secure and simple console interface

Tech Stack

Language: C#
