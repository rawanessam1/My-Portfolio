E-commerce Frontend Project (MoDance)

Overview

A responsive e-commerce frontend built with HTML, CSS, JavaScript, and Bootstrap, integrated with a live API for real product data.

Features

  • User authentication (login, register, forget password, reset password, OTP)
  • Product listing and product details pages
  • Shopping cart and checkout process
  • View past orders
  • Responsive design using Bootstrap
  • Integrated with a live E-commerce API for dynamic data

Tech Stack

  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap 5
  • External E-commerce API
