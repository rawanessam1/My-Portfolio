Todo-List React Project

Overview

A modern task management application built with React.js, Tailwind CSS, Ant Design, and React Router. Users can sign in, sign up, sign out, and manage tasks for today or the upcoming week. The app also includes a calendar view and separate pages for Today, Upcoming, and task management.

Features

  • User authentication: Sign in, Sign up, Sign out
  • Task management: Add, edit, delete tasks
  • Task categorization: Today, Upcoming Week
  • Calendar view to see tasks by date
  • React Router for smooth page routing
  • Persistent data using local storage
  • Responsive UI with Tailwind CSS and Ant Design components

Tech Stack

  • React.js
  • Tailwind CSS
  • Ant Design
  • React Router
  • Local Storage for persistent tasks
